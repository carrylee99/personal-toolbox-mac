# 个人工具箱安装使用说明

适用于 macOS。当前版本：`0.1.3`。

个人工具箱目前主要包含：

- 备忘录：记录待办、快速备忘、完成状态。
- 冒烟记录：按版本、场景、Case、步骤记录冒烟测试结果。
- 快捷键：支持快速记录备忘录、打开/隐藏主窗口。

数据默认保存在你自己选择的本地目录里。朋友安装后不会看到你的数据，需要他自己在设置里选择数据目录。

## 方式一：下载 DMG 安装使用

适合不想折腾代码、只想直接使用的人。

### 1. 下载安装包

打开 GitHub Release 页面：

[https://github.com/carrylee99/personal-toolbox-mac/releases/tag/v0.1.3](https://github.com/carrylee99/personal-toolbox-mac/releases/tag/v0.1.3)

下载这个文件：

`personal-toolbox-0.1.3.dmg`

### 2. 安装应用

1. 双击打开 `personal-toolbox-0.1.3.dmg`。
2. 把 `个人工具箱.app` 拖到 `Applications` / `应用程序` 目录。
3. 打开 `Applications` / `应用程序` 里的 `个人工具箱`。

### 3. 如果提示“已损坏，无法打开”

这是因为当前安装包没有 Apple Developer ID 签名和公证，macOS 会拦截从浏览器下载的应用。

可以在终端执行下面命令解除隔离属性：

```bash
xattr -dr com.apple.quarantine ~/Downloads/personal-toolbox-0.1.3.dmg
```

如果已经拖到应用程序目录后仍然打不开，再执行：

```bash
sudo xattr -rd com.apple.quarantine /Applications/个人工具箱.app
open /Applications/个人工具箱.app
```

执行 `sudo` 命令时，终端会要求输入本机开机密码。输入时不会显示字符，这是 macOS 的正常行为。

### 4. 第一次打开后的设置

进入左侧 `设置`，按下面顺序配置：

1. `常规`：选择一个长期使用的数据根目录。
2. `常规`：选择主题样式，比如 `清新自然` 或 `经典蓝色`。
3. `常规`：设置快捷键，比如快速备忘、打开主窗口。
4. `备忘录`：查看备忘数据路径，按需导入或导出备忘录。
5. `冒烟记录`：查看冒烟数据路径，按需设置 Markdown 输出目录。

推荐数据根目录示例：

```text
~/Documents/个人工具箱数据
```

不要选择临时目录，也不要选择下载目录，避免以后误删。

## 方式二：下载源码本地运行

适合愿意装 Node.js，或者 DMG 因为 macOS 安全策略打不开的人。

### 1. 安装 Node.js

先安装 Node.js LTS 版本：

[https://nodejs.org/](https://nodejs.org/)

安装完成后，在终端确认版本：

```bash
node -v
npm -v
```

能正常输出版本号就可以继续。

### 2. 下载源码

```bash
git clone https://github.com/carrylee99/personal-toolbox-mac.git
cd personal-toolbox-mac
```

### 3. 安装依赖

```bash
npm install
```

### 4. 启动应用

```bash
npm run dev
```

启动后会打开个人工具箱窗口。第一次使用同样需要进入 `设置`，选择数据根目录。

### 5. 下次继续使用

以后进入项目目录，执行：

```bash
npm run dev
```

## 常用功能

### 快速备忘

1. 在 `设置` 里配置 `快速备忘` 快捷键。
2. 按下快捷键后，会弹出一个很轻的小输入框。
3. 输入内容后按 `Cmd + Enter` 保存。
4. 按 `Esc` 关闭，不保存。

### 打开主窗口

1. 在 `设置` 里配置 `打开主窗口` 快捷键。
2. 按下快捷键可以打开或隐藏主窗口。
3. 再次打开时，会停留在上一次使用的页面。

### 备忘录

备忘录支持：

- 新增备忘。
- 编辑标题和详情。
- 标记完成或恢复未完成。
- 导入备忘录。
- 导出备忘录。

### 冒烟记录

冒烟记录支持：

- 创建业务版本。
- 每个版本下创建场景。
- 每个场景下创建 Case。
- 每个 Case 下维护多个步骤。
- 标记步骤成功、失败或未执行。
- 导入 Run Plan。
- 导出当前版本的冒烟记录 Markdown。

## 数据说明

应用数据保存在你选择的数据根目录里。

常见数据包括：

- 备忘数据。
- 冒烟记录数据。
- 冒烟记录导出的 Markdown。

换电脑使用时，可以把整个数据根目录复制过去，然后在新电脑的 `设置` 里重新选择这个目录。

## 简单排查

### 打开后显示 Vault 未连接

进入 `设置` -> `常规`，重新选择数据根目录并保存。

### 创建备忘失败

优先检查：

1. 是否已经选择数据根目录。
2. 目录是否存在。
3. 当前用户是否有这个目录的读写权限。

### 朋友安装后看不到我的数据

这是正常的。数据不会打进安装包里。

如果希望朋友使用同一份数据，需要把你的数据根目录单独复制给他，或者放到双方都能同步的目录里。

### DMG 安装和源码运行怎么选

- 普通使用：优先用 DMG。
- macOS 一直拦截安装包：可以用源码运行。
- 想自己改代码：用源码运行。
